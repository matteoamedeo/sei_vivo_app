# Style Guide

## Typography

### Line Height

La regola per il calcolo del `lineHeight` è:

```
lineHeight ≈ fontSize * 1.2 / 1.3
```

In pratica, il line height dovrebbe essere approssimativamente tra 1.2x e 1.3x la dimensione del font.

**Esempi:**
- `fontSize: 16` → `lineHeight: 20` (16 * 1.25)
- `fontSize: 20` → `lineHeight: 24` (20 * 1.2) o `lineHeight: 26` (20 * 1.3)
- `fontSize: 24` → `lineHeight: 30` (24 * 1.25)
- `fontSize: 32` → `lineHeight: 38` (32 * 1.2) o `lineHeight: 42` (32 * 1.3)

Questa regola garantisce una leggibilità ottimale e uno spacing coerente tra le righe di testo.
